Web Design

Web design is the process of planning and arranging content on a website so that it can be shared and accessed online with the world1. It is a type of digital design that determines the look of a website such as its colors, fonts, graphics, and user interface1. A web designer works on the appearance, layout, and content of a website2. A good web design is easy to use, aesthetically pleasing, and suits the user group and brand of the website2. Web design also includes web apps, mobile apps, and user interface design.
